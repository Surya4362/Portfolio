# Performance Marketing Portfolio

A personal portfolio site for a performance marketer specializing in lead generation, paid ads, outreach automation, and email marketing.

## 🔗 Live Site

> Replace this with your GitHub Pages URL after publishing:
> `https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## 📁 Structure

```
portfolio/
└── index.html   ← entire site (single file, no dependencies)
└── README.md
```

## 🚀 Deploying to GitHub Pages

1. Create a new GitHub repository (e.g. `portfolio`)
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)` → Save
5. Your site will be live at `https://YOUR-USERNAME.github.io/portfolio/` within a minute

## ✏️ Customizing

Before publishing, update these placeholders in `index.html`:

| Placeholder | What to replace with |
|---|---|
| `hello@yourdomain.com` | Your real email |
| `linkedin.com/in/yourprofile` | Your LinkedIn URL |
| `wa.me/91XXXXXXXXXX` | Your WhatsApp number |
| `PM` (avatar initials) | Your actual initials |
| `© 2025` | Current year |
